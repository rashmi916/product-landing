# 🎧 SoundWave X – Product Landing Page

A modern **Product Landing Page** built using **HTML and CSS**.
This project showcases a clean layout for promoting a product (SoundWave X Headphones) with sections like features, specifications, reviews, FAQ, and contact form.

---

## 🚀 Project Overview

The website is designed to present **SoundWave X Noise Cancelling Headphones** in a structured and user-friendly way.

It includes all key sections typically used in professional product landing pages.

---

## ✨ Features

* Hero Section with background image
* Product Features section
* Why Choose Us / Benefits section
* Product Specifications
* Customer Reviews
* Frequently Asked Questions (FAQ)
* Contact Form
* Footer section

---

## 🛠 Technologies Used

* HTML5
* CSS3
* Flexbox Layout
* Responsive Section Design
